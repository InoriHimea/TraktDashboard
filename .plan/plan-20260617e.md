# Plan 20260617e — Jellyfin 集成（電視節目看完刪除）

## 背景

集成 Jellyfin 的唯一目的：**看完電視節目後自動或手動從磁盤刪除分集文件**。

- 播放同步、播放跳轉由 Jellyfin 插件負責，本次不做
- 自動刪除需在設置中配置哪些媒體庫允許（主要為新番入庫庫）
- 未配置的庫只允許手動刪除
- 刪除按鈕需二次確認
- 架構：自部署單用戶，Jellyfin Admin API Key 直連

## 詳細設計

### E1 — DB Schema

在 `userSettings` 表新增 3 個欄位：

```sql
jellyfin_url        TEXT           -- Jellyfin 服務器地址，如 http://nas:8096
jellyfin_api_key    TEXT           -- Jellyfin Admin API Key
jellyfin_auto_delete_library_ids  TEXT  -- JSON 數組，存放允許自動刪除的庫 ID
```

### E2 — Migration

文件名：`0011_jellyfin_settings.sql`

```sql
ALTER TABLE "user_settings"
  ADD COLUMN "jellyfin_url" text,
  ADD COLUMN "jellyfin_api_key" text,
  ADD COLUMN "jellyfin_auto_delete_library_ids" text;
```

### E3 — TypeScript 類型

`UserSettings` 新增：

```ts
jellyfinUrl: string | null;
jellyfinApiKey: string | null;
jellyfinAutoDeleteLibraryIds: string[] | null;
```

新增響應類型：

```ts
export interface JellyfinLibrary {
    id: string;
    name: string;
    collectionType: string; // 'tvshows', 'movies', etc.
}

export interface JellyfinEpisode {
    id: string; // Jellyfin ItemId
    name: string; // 集標題
    seriesName: string; // 劇集名
    path: string | null; // 文件路徑（顯示用）
}
```

### E4 — Jellyfin API 路由

三個端點：

**GET /api/jellyfin/libraries**

- 向 Jellyfin GET `/Library/VirtualFolders` 轉發
- 返回 `{ data: JellyfinLibrary[] }`
- 若未配置 jellyfinUrl/jellyfinApiKey 返回 503

**GET /api/jellyfin/episode/:showTmdbId/:season/:episode**

- 向 Jellyfin `GET /Items` 查詢，篩選：
    - `IncludeItemTypes=Episode`
    - `Recursive=true`
    - `TmdbId=<showTmdbId>` （通過 ProviderIds 匹配）
    - `Season=<season>` + `ParentIndexNumber=<episode>` 篩選
    - 策略：先按 show tmdbId 拿列表，再在結果中按 seasonNumber + episodeNumber 比對
- 返回 `{ data: JellyfinEpisode | null }`

**DELETE /api/jellyfin/items/:jellyfinItemId**

- 向 Jellyfin `DELETE /Items/<jellyfinItemId>?deleteFiles=true` 發送請求
- 返回 `{ ok: true }`

### E5 — 路由注冊

在 `apps/api/src/index.ts` 添加：

```ts
import { jellyfinRoutes } from "./routes/jellyfin.js";
app.route("/api/jellyfin", jellyfinRoutes);
```

### E6 — Settings API 更新

`GET /api/settings` 返回結果加入 jellyfin 字段（API key 不返回明文，返回是否已設置的布爾值供 UI 顯示 masked）。  
實際上為了 UI 配置方便，Settings 頁需要回填 API key，故全部返回。

`POST /api/settings` 驗證並保存新的 jellyfin 字段：

- `jellyfinUrl`：可選，需為合法 URL（http/https）
- `jellyfinApiKey`：可選字符串
- `jellyfinAutoDeleteLibraryIds`：JSON 字符串數組

### E7 — 自動刪除 Hook

在 `POST /api/shows/:showId/episodes/:season/:episode/watch` 的 `recalcShowProgress` 後添加：

```ts
// Jellyfin auto-delete (fire-and-forget; errors must not fail the watch record)
try {
    const settings = await db.select().from(userSettings).where(eq(userSettings.userId, userId));
    const cfg = settings[0];
    if (cfg?.jellyfinUrl && cfg?.jellyfinApiKey && cfg?.jellyfinAutoDeleteLibraryIds) {
        const autoDeleteIds: string[] = JSON.parse(cfg.jellyfinAutoDeleteLibraryIds);
        if (autoDeleteIds.length > 0) {
            // 查找 Jellyfin 分集並刪除（若屬於已配置庫）
            await autoDeleteJellyfinEpisode(
                cfg.jellyfinUrl,
                cfg.jellyfinApiKey,
                autoDeleteIds,
                show,
                ep,
            );
        }
    }
} catch {
    // 自動刪除失敗不影響記錄觀看記錄
}
```

`autoDeleteJellyfinEpisode` 提取到 `apps/api/src/services/jellyfin.ts`。

## Jellyfin API 說明

| 用途            | Jellyfin 端點                                                                                                                    |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| 拉取媒體庫列表  | `GET /Library/VirtualFolders`                                                                                                    |
| 按 TMDB ID 查集 | `GET /Items?IncludeItemTypes=Episode&Recursive=true&ParentIndexNumber=<S>&IndexNumber=<E>&AnyProviderIdEquals=Tmdb.<showTmdbId>` |
| 刪除文件        | `DELETE /Items/<itemId>?deleteFiles=true`                                                                                        |

認證統一通過 Header：`X-Emby-Token: <apiKey>`
